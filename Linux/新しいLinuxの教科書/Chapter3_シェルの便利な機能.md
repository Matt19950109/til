## プロンプトとコマンドライン
- コマンドライン → プロンプト記号の後ろに入力するコマンド入力部分のこと

## Ctrlキーをうまく使う

- Ctrl + b → 後方に1文字分移動
- Ctrl + f → 前方に1文字分移動
- Ctrl + a → 行頭に移動
- Ctrl + e → 行末に移動

## 単語単位でのカーソル移動
- Meta(Alt) + b → 後方に単語一つ分移動
- Meta(Alt) + f → 前方に単語一つ分移動

## 文字の削除
- BackSpaceキー or Ctrl + h → カーソル位置の後方に1文字削除
- Deleteキー or Ctrl + d → カーソル位置に重なった1文字の削除
- Ctrl + w → 後方にスペース区切りで1単語文削除
