# エイリアス
分かりやすい別名を与えてコマンドを使いやすくするための機能

    alias <名前>='<コマンド>'
    
    例）
    alias la='ls -a'
    alias cp='cp -i'

## エイリアスの確認と削除

    確認コマンド
    type <コマンド>
    
    削除
    unalias <コマンド>

### 一時的にエイリアスを無効にしたい場合
    ①フルパスで指定する
    /bin/ls
    
    ②command <コマンド>を使う
    command ls
    
    ③\を先頭につける
    \ls
