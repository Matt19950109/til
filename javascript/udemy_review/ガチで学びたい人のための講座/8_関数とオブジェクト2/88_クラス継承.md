## クラス継承

- ほかのクラスのプロパティとメソッドを継承すること

    class Person {
      constructor(name, age) {
        this.name = name;
        this.age = age;  
      }
    }
  
    class Japanese extends Person {
      constructor(name, age, gender)
        super(name, age);
        this.gender = gender; 
      }
    
      hello() {
      console.log('hello' + this.name);
      }
    }
