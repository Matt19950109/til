## 4-01 onsubmitイベント
フォームの送信ボタンをクリックしたときのイベント

    取得した要素.onsubmit = finction() {
      //処理内容
    };
    ※右辺のファンクションに名前はつけず。{～}内にreturnは不要

- action属性…データを送信する先のURLを指定する
- type属性…フォーム部品の種類を決める
- name属性…入力されたデータのサーバー送信時、そのデータに付与される名前となる
