## head
- ページのメタ的の情報を記載する(文字コードなど)

- 空タグ → 終了タグのないタグのこと

## ブロック要素とインライン要素
- ブロック要素  
  情報を上下に並べるときに使用

- インライン要素
  情報を横に並べるときに使用

- タグの種類によってインラインとブロックの要素を持っているかが異なる

      クラスの属性値を持っているタグを指定して装飾する
      .first {
      
      }
      
      idの属性値を持っている場合
      #first{
      
      }
